# 不挠的舰载机 - 快速开始指南

## Phase 1 MVP 版本使用说明

### 如何使用

#### 1. 获取物品
在创造模式中，打开物品栏找到 **"不挠的舰载机"** 标签页，获取：
- **水平轰炸机召唤器**

#### 2. 设置目标
1. 将召唤器放在**副手**（左手）
2. 看向你想轰炸的目标位置
3. **右键**使用副手物品
4. 你会看到消息：`§e已锁定目标: (X, Y, Z)`

#### 3. 召唤飞机
1. 将召唤器切换到**主手**（右手）
2. **右键**使用
3. 一架轰炸机会在你头顶20格高度出现
4. 你会看到消息：`§a已召唤水平轰炸机`

#### 4. 观察任务执行
飞机会自动执行以下流程：
1. **待命** - 在你头顶盘旋
2. **赶赴** - 飞向目标上方（80格高度）
3. **投弹** - 每1秒投下一枚炸弹（共6发）
4. **返航** - 弹药耗尽后飞回你的位置并消失

### 操作技巧

#### 锁定地面目标
- 看向地面某个方块
- 副手右键锁定
- 飞机会飞到该位置正上方投弹

#### 锁定远距离目标
- 可以锁定最远200格的目标
- 超过此距离会显示 `§c未找到目标`

#### 召唤多架飞机
- 可以连续召唤多架飞机攻击同一目标
- 所有飞机会共享当前锁定的目标

### 炸弹参数

- **爆炸半径**: 4格
- **伤害**: 30点（15颗心）
- **弹药数量**: 6发/架
- **投弹间隔**: 1秒

### Phase 1 限制

当前版本是最小可玩版本（MVP），存在以下限制：

❌ 无自定义模型（飞机仅显示阴影，炸弹显示为TNT物品）  
❌ 无音效和粒子效果  
❌ 无降落动画（飞机返航后直接消失）  
❌ 无编队系统  
❌ 无控制面板GUI  
❌ 每个玩家只能锁定一个目标  

这些功能将在后续Phase中逐步实现。

### 故障排查

#### 飞机不飞向目标
- 确认已用副手锁定目标
- 查看聊天栏是否显示 `已锁定目标` 消息

#### 飞机消失了
- 检查是否6发炸弹已全部投下（飞机会自动返航并消失）
- 确认玩家仍在线（飞机绑定到召唤玩家）

#### 炸弹不爆炸
- 炸弹需要接触地面或实体才会爆炸
- 从80格高度落下需要约4秒

### 开发测试建议

1. **在平坦地形测试** - 便于观察飞行轨迹
2. **使用F3+B** - 显示实体碰撞箱
3. **旁观者模式** - 跟随飞机观察投弹过程
4. **记录坐标** - 对比锁定坐标和实际命中点，评估精度
