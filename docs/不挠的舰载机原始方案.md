## 零、子系统总览

### 子系统划分

**核心执行层**

```
飞机实体系统
└── 基类、参数变量接口、双武器槽、飞行物理

状态机系统
└── 状态定义、切换、客户端同步

弹道系统
└── 三坐标计算、各弹药弹道、命中偏差
```

**决策控制层**

```
锁定子系统
└── 自动锁定档位、目标优先级、对接火控

火控系统
└── 玩家/长机按键锁定、多目标列表、目标分配模式

编组系统
└── 小中大队管理、任务过滤、等齐出发、对空模式
```

**特殊功能层**

```
长机与区块管理系统
└── 长机指定/转移、强制区块加载、多长机视角切换

反潜探测系统
└── 水下目标高亮、探测范围、双视角显示

弹药管理系统
└── 弹匣合并、投弹配置、补给（待设计）
```

**界面层**

```
控制面板（终端）
└── 地图、配置、指挥、弹药装载管理
    以长机为中心、多长机切换显示
```

### 依赖关系

```
控制面板 -> 所有系统
火控系统 -> 锁定子系统 -> 状态机 -> 弹道系统
编组系统 -> 长机系统
长机系统 -> 侦察视角（内含）
```

## 一、机型分类

所有飞机采用**双武器槽**设计，机型能力完全数据驱动，新增机型无需改动代码。飞机基础属性（速度、HP、载弹量等）全部通过变量接口暴露，由配置表填写，代码不硬编码。

### 武器槽结构

| 槽位 | 允许弹药 | 说明 |
|------|----------|------|
| 对空槽 | 弹匣 | 前射或自卫炮塔，由机型配置决定 |
| 对海槽 | 航弹 / 鱼雷 / 火箭弹 | 装什么弹药触发对应攻击行为 |

### 对空能力类型

- **前射**：主动向前方开火，用于空战/护航
- **自卫炮塔**：被动防御，以飞机为中心的球形自卫范围内有目标时触发反击
- **两者兼有**：如 B-25 前射 + 自卫炮塔
- **无**：部分机型无对空能力

### 对海攻击行为（由弹药类型决定）

| 弹药 | 攻击行为 | 前提 |
|------|----------|------|
| 航弹 | 水平轰炸 | 机型支持水平投弹 |
| 航弹 | 俯冲轰炸（制导） | 机型支持俯冲 |
| 鱼雷 | 鱼雷攻击 | — |
| 火箭弹 | 火箭齐射 | — |

> 水平轰炸和俯冲轰炸都使用航弹，但没有既能俯冲又能水平的机型，由机型配置唯一决定。

### 机型举例

| 机型 | 对空槽 | 对海槽允许弹药 | 攻击模式 |
|------|--------|----------------|----------|
| B-25 | 前射 + 自卫炮塔 | 航弹 | 水平轰炸 |
| BTD | 前射 | 航弹 / 鱼雷 | 俯冲轰炸 / 鱼雷 |
| 反潜机 | — | 航弹 | 水平轰炸，目标类型=水下 |

## 二、弹药系统

### 弹药种类

| 弹药 | 物品形式 | 备注 |
|------|----------|------|
| 子弹 | 弹匣（1000发/个） | 支持合并 |
| 航空炸弹 | 单枚 | 水平轰炸 / 俯冲轰炸 |
| 航空鱼雷 | 单枚 | 鱼雷攻击 |
| 火箭弹 | 单枚 | 火箭齐射 |

### 弹匣合并规则

```
合并结果 = A发 + B发
  <= 1000  ->  一个弹匣，A+B 发
  >  1000  ->  一个满弹匣（1000发），溢出丢弃
```

玩家可选择不合并以避免溢出，飞机允许不满弹起飞。

### 投弹配置（玩家全局设置）

| 配置项 | 类型 | 说明 |
|--------|------|------|
| 单次投弹数量 | 数量 | 每次飞越目标投弹数，所有机型共用 |
| 单次最小有效伤害 | 标称伤害 | 剩余弹药标伤低于此值时全弹投出 |

**投弹判断逻辑：**

```
剩余弹药标称总伤害 >= 最小有效伤害
  -> 正常投"单次投弹数量"颗

剩余弹药标称总伤害 < 最小有效伤害
  -> 全部投完（不值得单独跑这趟）
```

**示例：** 4载弹量鱼雷机，单次投3，最小有效伤害=2颗标伤

- 第一轮：剩4颗，满足条件，投3颗
- 第二轮：剩1颗，低于门槛，全投1颗

**中队协同投弹：** 中队集体出击时，同中队的鱼雷机一次全投，形成饱和攻击。

## 三、弹道与命中机制

### 三坐标体系

```
目标坐标     每 tick 更新，跟随目标实体
    |
    |  目标速度 × 武器预计飞行时间
    v
弹着点       锁定时计算，可选动态修正
    |
    |  飞机速度 / 高度 / 武器弹道反推
    v
投弹坐标     飞机需要抵达的位置（APPROACH 的终点）
```

弹着点计算封装在武器内，不同武器有不同飞行时间参数。

### 各弹道类型

| 类型 | 机型 | 命中特点 | 实现方式 |
|------|------|----------|----------|
| 自由落体 | 水平轰炸机、反潜机 | 命中一般，总伤高 | 投弹时对弹着点加随机偏移 |
| 制导 | 俯冲轰炸机 | 命中高，伤害一般 | 弹药实体持续追踪目标，有锁定丢失概率 |
| 自走（鱼雷） | 鱼雷机 | 命中差，伤害极高 | 发射时线性外推方向，之后不干预，目标可躲避 |
| 自走（火箭） | 火箭机 | 命中介于制导与鱼雷之间 | 直线弹道，发射后不追踪 |

## 四、飞机状态机

### 状态列表

| 状态 | 中文 | 说明 |
|------|------|------|
| STANDBY | 待命 | 在玩家头顶跟随盘旋 |
| ORBITING | 盘旋待命 | 在指定盘旋点等待指令 |
| APPROACH | 赶赴 | 飞向目标坐标/投弹坐标/转向点 |
| CIRCLING | 盘旋 | 通用盘旋行为 |
| LOCKED | 锁定 | 调整航线，计算三坐标 |
| ATTACKING | 打击中 | 投弹前的攻击航段 |
| DROPPING | 投弹 | 实际投弹动作 |
| POST_ATTACK | 确认打击效果 | 飞越目标后确认死亡 |
| DOGFIGHT | 空战 | 前射型主动追击空中目标 |
| RETURNING | 返航 | 弹药耗尽返回玩家 |

> 无燃料消耗，RETURNING 仅在弹药耗尽时触发。

### 状态接口（Java）

```java
interface IPlaneState {
    void onEnter(PlaneEntity plane);
    void onTick(PlaneEntity plane);
    void onExit(PlaneEntity plane);
}
```

挂载在实体 `aiStep()` 中，状态切换时触发 onExit/onEnter，动画与逻辑模块解耦。

## 五、空袭攻击流程

```
[锁定子系统] 给出目标
    |
[计算三坐标] 目标坐标 -> 弹着点 -> 投弹坐标
    |
[APPROACH] 赶赴投弹坐标
    |
[DROPPING] 投出"单次投弹数量"颗弹药
    |
[检查弹药]
  ├── 不足 -> [RETURNING]
  └── 充足
        |
      [APPROACH] 赶赴转向点（飞越目标，自然拉起）
        |
      [POST_ATTACK] 确认目标是否死亡
        ├── 死亡 -> 锁定子系统重新索敌
        │     ├── 找到新目标 -> 回到[计算三坐标]
        │     └── 无目标    -> [RETURNING]
        └── 存活 -> 重新计算三坐标
              |
            回到 [APPROACH] 赶赴投弹坐标
```

**转向点**：投弹时飞行方向的延伸距离，自动计算，距离为机型配置参数。

## 六、空战流程

### 前射型主动空战（DOGFIGHT）

```
发现空中目标（主动模式下主动搜索，一般模式下被攻击时触发）
    |
[APPROACH] 追击目标
    |
    追上（进入额定交战距离）
    |
减速保持额定距离，持续射击
    |
    满足放弃条件？
    ├── 目标速度 > 本机速度（追不上）-> 放弃
    ├── 目标超出编组模拟距离范围    -> 放弃
    └── 目标消灭                    -> 重新索敌或返回待命
```

### 自卫炮塔（被动）

- 触发条件：以飞机为中心的**球形自卫范围**内出现空中目标
- 触发方式：一般模式下被攻击才触发；主动模式下范围内即触发
- 低攻击性模式：完全不触发

## 七、锁定子系统

锁定子系统独立于飞机状态机，对外提供目标和三坐标。

### 优先级

```
1. 火控系统指定目标（最高优先）
2. 自动锁定模式
```

### 自动锁定档位

| 档位 | 说明 |
|------|------|
| NEAREST | 最近目标 |
| STRONGEST | 威胁值最高 |
| FOCUS | 集火，与其他飞机打同一目标 |
| SPREAD | 分散，多机自动分配不同目标 |
| TYPE_FILTER | 按目标类型优先（仅多用途机可配置） |

## 八、火控系统

### 锁定方式

玩家或长机视角下，按指定按键锁定准星所指目标，支持叠加锁定多个目标，形成火控目标列表。

### 目标分配模式

| 模式 | 行为 |
|------|------|
| 集火模式 | 所有飞机打火控列表第一个目标 |
| 普通模式 | 飞机按目标数量循环分配（如5架飞机、3个目标，则按1/2/3/1/2分配） |

分配模式可在控制面板中配置。

### 与锁定子系统的关系

```
火控有指定目标  ->  锁定子系统使用火控目标列表
火控无指定目标  ->  锁定子系统按自动档位工作
```

## 九、编组系统

编组以**中队**为核心，小队和大队为辅助管理层级。

### 层级结构

```
大队（混编）
└── 统一调度目的地 / 盘旋点，不干涉攻击细节
    |
    中队（同攻击类型）
    ├── 对空中队    所有飞机有对空能力
    ├── 鱼雷中队    所有飞机挂鱼雷
    ├── 轰炸中队    航弹机型
    └── 反潜中队    对水下目标
        |
        小队（同机型）
        └── 中队的子单位，方便细化管理
```

### 等齐出发

玩家发出出击指令时，以**所选编队层级**为单位等齐再出发：

```
选择大队出击  ->  大队内所有飞机齐了再走
选择中队出击  ->  该中队所有飞机齐了再走
选择小队出击  ->  该小队所有飞机齐了再走
```

等待期间飞机在各自当前位置（盘旋点或玩家头顶）保持 ORBITING/STANDBY 状态。

### 大队任务参与过滤

发布攻击指令时，自动按目标类型筛选参与中队：

```
攻击水面目标  ->  有对海能力的中队参与
                  反潜中队、纯战斗机中队跳过

攻击空中目标  ->  有对空能力的中队参与
                  反潜中队、无对空槽机型跳过
```

### 大队对空模式（三档）

此设置仅控制**对空行为**，对海攻击流程不受影响。

| 模式 | 前射型 | 自卫炮塔型 |
|------|--------|------------|
| 一般模式（自卫） | 被攻击时追击还击 | 自卫球形范围内被攻击才触发 |
| 主动模式（防空） | 主动搜索并拦截空中目标 | 靠近空中目标，在自卫范围内持续输出 |
| 低攻击性模式（管控） | 只打火控指定的空中目标，不还击 | 同左，完全忽视未指定的空中威胁 |

## 十、长机与区块管理系统

### 长机职责

长机兼任**侦察视角锚点**和**区块强制加载锚点**，使飞机编队可在超出玩家视距的范围内正常运作。

### 长机选择规则

```
默认  ->  编组中自动选一架担任长机
手动  ->  玩家在控制面板中指定
击落  ->  自动转移给编组中下一架飞机
```

### 侦察视角

- 玩家可随时切入任意长机的第一人称视角
- 可在多架长机之间自由切换
- 可随时按键切回舰娘（玩家实体）视角
- 长机被击落时强制切回玩家视角
- 长机视角中可以使用火控按键锁定目标，效果与玩家视角相同
- 长机**不能攻击**，纯侦察功能

### 控制面板显示

- 地图以当前选中长机为中心
- 显示当前所有强制加载区块的数量和位置
- 可在多个长机之间切换，地图中心随之切换

## 十一、反潜探测系统

- 反潜机在飞行时对一定范围内的水下目标进行高亮显示
- 高亮在**玩家视角**和**长机视角**中均可见
- 不在控制面板地图上标注
- 探测范围为机型配置参数

---

## 十二、盘旋点

- 玩家在控制面板地图上手动指定的固定坐标
- 部署到盘旋点后飞机进入 ORBITING 状态待命
- 收到攻击指令后切换至 APPROACH
- ORBITING 与 STANDBY（玩家头顶）是两个独立的待命状态

---

## 十三、控制面板（终端物品）

可手持使用的物品，打开后展示 GUI 界面。

### 功能分层

**显示层**
- 以当前选中长机为中心的小地图
- 各飞机位置（不同机型不同图标）
- 当前飞行航线
- 打击目标标记
- 强制加载区块数量与位置
- 多长机切换入口

**配置层**
- 单次投弹数量
- 单次最小有效伤害
- 自动锁定档位
- 大队对空模式
- 火控目标分配模式（集火/普通）
- 多用途机 TYPE_FILTER

**指挥层**
- 手动指定打击坐标
- 设置/移动盘旋点
- 召回所有飞机
- 长机手动指定
- 弹药装载管理（各飞机弹药槽统一管理）
