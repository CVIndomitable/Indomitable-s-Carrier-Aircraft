## 🚀 启动测试

### 启动命令

```bash
# 方式1：Gradle启动（开发环境）
./gradlew runClient

# 方式2：直接运行jar（需要安装NeoForge）
# 1. 复制 build/libs/indomitablecarrieraircraft-0.1.0-dev.jar
# 2. 放入 .minecraft/mods/ 目录
# 3. 使用NeoForge 1.21.1启动
```

### 测试步骤

1. **进入创造模式**
   ```
   /gamemode creative
   ```

2. **获取召唤器**
   - 打开物品栏 (E键)
   - 切换到"不挠的舰载机"标签页
   - 拿取"水平轰炸机召唤器"

3. **锁定目标**
   - 召唤器放副手 (F键交换)
   - 看向要轰炸的位置
   - 右键锁定

4. **召唤飞机**
   - 召唤器放主手
   - 右键召唤
   - 抬头观察飞机

5. **观察任务执行**
   - 使用F5切换第三人称
   - 观察飞机飞向目标
   - 等待投弹完成
   - 观察返航

### 调试技巧

```
F3        - 显示调试信息（实体数量、坐标）
F3+B      - 显示碰撞箱（看到飞机和炸弹的范围）
F3+G      - 显示区块边界
F1        - 隐藏HUD（更好观察）
/kill @e[type=indomitablecarrieraircraft:bomber_aircraft]  - 清除所有飞机
```
