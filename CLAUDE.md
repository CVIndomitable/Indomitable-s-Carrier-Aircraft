# 不挠的舰载机模组

## 项目信息

- **MC 版本**: 1.21.1
- **加载器**: NeoForge
- **状态**: 方案评估阶段
- **创建日期**: 2026-06-02
- **预估工作量**: 5-8 个月全职开发

## 项目目标

实现一个功能完整的舰载机系统，核心特性：
- AI 控制的飞机实体（双武器槽，数据驱动）
- 复杂状态机（待命、赶赴、攻击、空战、返航等）
- 多种弹道类型（自由落体、制导、鱼雷、火箭）
- 火控与锁定系统
- 三级编组系统
- 长机视角与区块加载
- 控制面板 GUI

## 开发规范

### 代码架构
- 所有机型配置使用 JSON 数据包
- 状态机使用接口 + 枚举模式
- 弹道计算封装在独立的工具类
- 网络包使用 NeoForge 的 SimpleChannel
- 客户端渲染使用事件总线

### 包结构
```
com.yourmod.indomitable
├── entity/              # 飞机实体
├── ai/                  # AI 与状态机
├── ballistic/           # 弹道计算
├── firecontrol/         # 火控系统
├── formation/           # 编组系统
├── gui/                 # 控制面板
├── network/             # 网络同步
└── util/                # 工具类
```

## 风险提示

⚠️ 这是一个高复杂度项目，主要技术挑战：
1. **AI 路径规划**（最高难度）- 三坐标计算与飞行物理
2. **性能优化** - 多架飞机 + 区块加载
3. **GUI 开发** - 实时地图渲染
4. **网络同步** - 复杂状态的客户端同步

> 📚 **详细分析**: 参见 [技术挑战分析](docs/技术挑战分析.md) 和 [风险与简化方案](docs/风险与简化方案.md)
